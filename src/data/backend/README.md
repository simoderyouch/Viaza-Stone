# Legacy catalogue data

The catalogue that was previously active on the website is retained here as a backend/archive source:

- `legacy-products.ts` contains the original product records.
- `legacy-collections.ts` contains the original collection and navigation records.

The active public catalogue is now defined in `src/data/products.ts` and `src/data/collections.ts`, using the supplied Viaza Stone catalogue extraction and its product/project assets.
